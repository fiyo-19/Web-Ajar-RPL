Bab 1 ALUR LOGIKA PEMROGRAMAN

A. Konsep Algoritma

Algoritma memiliki istilah berasal dari kata “algorism” yang merupakan panggilan popular bangsa Eropa terhadap seorang ahli matematika muslim dari Jazirah Arab Bernama Abu Ja’far Muhammad Ibnu Al-Khwarizmi yang hidup di abad ke-19. Namun dalam perkembangan, istilah “algorism” berubah menjadi “algorithm” karena keterkaitan dengan istilah dalam matematika. Jadi algorima merupakan urutan Langkah – Langkah logis penyelesaian masalah yang disusun secara sistematis dan logis. Ada beberapa software atau aplikasi yang mendukung algoritma dan pemrograman antara lain ada Turbo Pascal, Dev C++, CV AVR, NetBeans.

C. Struktur Algoritma dengan Natural Language

Penulisan menggunakan metode natural language merupakan pendekatan penulisan algoritma bergaya Bahasa manusia yang digunakan dalam keseharian. Dengan natural language memiliki sifat deskriptif, singkat, padat, dan mudah dimengerti, tetapi masih mewakili proses yang akan dilakukan. Metode penulisan algoritma menggunakan natural language memiliki keuntungan dan kerugian. Keuntungannya antara lain :

  • a. Mudah penulisannya
  • b. Sedehana
  • c. Tidak sulit dalam mempelajarinya karena tidak memiliki aturan standar dalam penulisan
  • d. Tidak membutuhkan tool atau software khusus dalam perancangannya

Adapun kerugiannya antara lain :

  • a. Gaya penulisan setiap programmer tidak sama
  • b. Pemahaman dalam pembacaan algoritma menjadi sulit
  • c. Sukar untuk menerjemahkannya ke dalam bentuk kode pemrograman.

E. Struktur Algoritma dengan Pseudocode

Pseudo memiliki arti tidak sebenarnya, semu, atau samar. Jadi pseudocode merupakan sebuah notasi kode yang tidak sebenarnya, tetapi mewakili logika algoritma Bahasa pemrograman yang akan digunakan. Namun, pseudocode tidak memiliki aturan penulisan khusus seperti dalam Bahasa pemrograman. Struktur algoritma dengan pseudocode memiliki pedoman penulisan sebagai berikut:

  1. Memiliki bagian header (kepala) yang menunjukkan judul algoritma, komentar, dan deklarasi
  2. Memiliki bagian badan algoritma yang merupakan inti dan proses.
  3. Memiliki bagian akhir proses algoritma yang menandakan bahwa proses alir program algoritma telah berakhir.
  4. Lebih banyak menggunakan bahasa inggris dalam mempresentasikan alir program
  5. Untuk menulis komentar diawali dengan karakter “{“ dan diakhiri dengan karakter “}”.

F. Mengenal Tipe Data

Tipe data adalah sebah karakter atau jenis dari objek yang memiliki jangkauan nilai memori dalam computer. Pemanggilan tipe data digunakan untuk memanipulasi sebuah objek yang diwakili oleh variabel dan kontanta. Variabel merupakan nama bentukan yang dibuat oleh pemrograman dengan ketentuan :

  • Diawali dengan karakter bukan angka numerik
  • Dapat berupa gabungan antara karakter dan angka numerik
  • Tidak boleh menggunakan spasi

1. Tipe Data Sederhana (Predefined Data Type )

a. Jenis Bilangan Bulat
Jenis bilangan blat memiliki karakteristik yaitu nilai numerik yang tidak mengandung pecahan decimal, seperti 5, 8, 20. Tipe data bilangan bulat ada byte , shortint, word, integer, longint

b. Jenis bilangan Real
Ciri dari bilangan ini yaitu bilangan yang mengandung nilai pecahan bentuk decimal, misalnya 2.30. Di dalam pemrograman nilai pecahan decimal diawali tanda titik “.” Setelah nilai bilangan bulat. Tipe data dari bilangan real ada single, double, real, extended.

c. Jenis bilangan logika
Tipe data dikenal sebagai nilai data Boolean yang hanya memiliki dua nilai data, yaitu true dan false . Nilai tue yaitu 1 dan false yaitu 0.

d. Jenis data Karakter
Semua karakter mulai abjad ‘a’..’z’, ‘A’..’Z’, angka ‘0..’9’, smapai symbol khusus seperti ‘!’,’#’,’$’,’%’,’^’ dst . Proses penetapan variabel memiliki nilai data karakter adalah selalu diapit dengan tanda petik satu.

e. Jenis data string
Tipe data ini merupakan deretan atau gabungan beberapa karakter sehingga membentuk sebuah kalimat.

2. Tipe Data Bentukan (user-defined data type)

Tipe ini merupakan sebuah tipe yang terbentuk dari gabungan beberapa tipe data sederhana seperti yang disebutkan sebelumnya. Ada dua jenis tipe data bentkan yaitu tipe data dengan nama baru dan tipe data terstruktur.

G. Mengenal Operator

Operator merupakan diibaratkan seorang pelayan yang bertugas melakukan sesuatu layanan teretntu terhadap pelanggan. Namun dalam algoritma operator yaitu sebuah symbol operasi tertent dalam melakukan sebuah operasi data. Sedangkan data yang dioperasikan oleh operator yaitu operand.

1. Operasi Terhadap Bilangan Bulat

Operasi aritmatika nya sama seperti dengan operasi terhadap bilangan bulat , akan tetapi tidak ada div dan mod . Kemudian untuk operasi perbandingan sama seperti operasi perbandingan di bilangan bulat.

3. Operasi Terhadap Bilangan Logika

Konsep dasar operasi ini merupakan proses komputasi dalam mesin computer yang akan dijelaskan di dalam Gerbang Logika di system komputer dengan menghasilkan nilai true (1) dan false (0).

a. Operator AND : menhasilkan nilai true jika kedua operand-nya bernilai true.

b. Operator NAND (NOT AND) : kebalikan dari operator AND.

c. Operator OR : menghasilkan nilai true jika salah satu atau semua operand bernilai true.

d. Operator NOR (NOT OR) : menghasilkan nilai ture jika kedua operandnya bernilai false.

e. Operator NOT : menghasilkan nilai true apabila operandnya bernilai false dan begitu sebaliknya.

f. Operator XNOR : menghasilkan nilai true jika kedua operandnya bernilai sama

g. Operator XOR : menghasilkan nilai true jika kedua operand nya berbeda nilai.
