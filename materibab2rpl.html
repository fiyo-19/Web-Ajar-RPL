A. Pengertian Bahasa Pemrograman

Secara umum Bahasa merupakan sebuah sarana sebagai berkomunikasi antara pengguna. Maksutnya adalah sebgai alat komunikasi manusia dengan komputer / PC. Program sendiri yaitu sebuah deretan perintah atau instruksi yang dikenal sebagai mesin komputer untuk mengerjakan suatu hal proses. Jadi Bahasa pemrograman adalah sebuah Bahasa yang dapat menerjemahkan perintah yang dapat diberikan oleh penguna dalam susunan instruksi berupa kode terstruktur kepada mesin komputer untuk mengerjakan suatu proses tertentu. Bahasa pemrograman memiliki karakteristik diantaranya :

  • 1. Dalam pendeklarasian memiliki tata Bahasa dengan aturan khusus
  • 2. Memiliki sebuah interrupt library sebagai menerjemahkan perintah yang diinputkan
  • 3. Menggunakan interpreter atau compiler sebagai menerjemahkan sintaks pemrograman ke dalam Bahasa mesin

Berdasarkan penggunannya, Bahasa pemrograman memiliki dua kategori sebagai berikut:

  • 1. Bahasa pemrograman sebagai tujuan umum, maksudnya dapat digunakan untuk menciptakan aplikasi yang bersifat umum atau general. Contohnya seperti gambar diatas yaitu ada java, C++, Delphi
  • 2. Bahasa pemrograman sebagai tujuan khusus, maksudnya dapat dibuat dengan keperluan tertentu. Misalnya pad COBOL untuk bisnis dan administrasi, FORTRAN sebagai keomunikasi ilmiah, dan Assembler sebagai pengontrol hardware.

Level Bahasa pemrograman dibagi menjadi dua yaitu

  • - Bahasa tingkat rendah (Low Level Language), yaitu bahasa yang hanya dimengerti oleh mesin berupa bilangan 0 dan 1 atau on-off. Bahasa ini langsung dapat dieksekusi mesin ketika diintruksikan pengguna kepada mesin tanpa melalui media translator. Contohnya yaitu Bahasa rakitan atau Assembler.
  • - Bahasa tingkat tinggi (High Level Language), yaitu Bahasa yang dibuat dengan mendekati Bahasa manusia sehari – hari. Apabila dieksekusi Bahasa tinkat tinggi membutuhkan compiler sebagai menerjemahkan ke dalam Bahasa mesin. Contohnya yaitu java, C, BASIC. Kemudian Bahasa ini wajib membutuhkan bantuan translator sebagai mengonversi sintaks oleh mesin komputer yang dikenal dengan istilah penerjemah Bahasa atau language translator.

Ada dua jenis language translator yaitu interpreter dan compiler. Intepreter yaitu sebuah kode program diterjemahkan baris demi baris hingga akhir baris program. Apabila terdapat erorr pada sebuah baris tertentu, baris program sebelumnya tetap akan diekseskusi. Sedangkan , compiler yaitu suatu aplikasi peng-compile program yang memeriksa semua baris kode program yang telah dideklarasikan sampai tidak terjadi erorr. Aplikasi yang menggunakan compiler yaitu COBOL, C dan Pascal.

B. Jenis Bahasa Pemrograman

1. COBOL (Common Business Oriented)

Yaitu sebuah Bahasa pemrograman tingkat tinggi yang lahirnya pada tahun 1959. Bahasa ini memiliki keahlian khusus dalam mempermudah design pembuatan aplikasi di bidang bisnis yang digunakan dalam komputer. Kelebihan dari Bahasa ini yaitu script pemrogramannya menggunakan keyword dalam Bahasa inggris sehingga cukup memudahkan programmer dalam mendesain aplikasi dan kemampuan yang baik dalam memberikan penanganan proses pada peranti masukan atau keluaran.Berikut contoh kode program COBOL

2. FORTRAN (Formula Translation)

Yaitu Bahasa yang diperkenalkan oleh John Backus di IBM pada tahun 1956. Bahasa ini digunakan dalam mendesain pembuatan aplikasi dalam bidang IPTEK, matematika, dan teknik. Bahasa ini juga masuk ke dalam Bahasa tingkat tinggi generasi pertama sebelum COBOL ada. Kelebihan dari FORTRAN adalah kemampuan dalam operasi aritmatika bahkan dalam suatu bilangan kompleks.

3. C

Yaitu Bahasa yang diorientasikan pada structural dan procedural. Bahasa ini dikembangkan pada tahun 1972 di Bell Laboratories oleh Dennis Ritchie. Bahasa ini masuk ke dalam Bahasa tingkat tinggi karena dapat di compiler dalam deklrasinya. Bahasa C sebagai system programming language dapat digunakan untuk melakukan low-level programming. Bahasa C sebagai procedural language menentukan beberapa langkah untuk program agar dapat menyelesaikan masalah.

4. PASCAL

Yaitu Bahasa pemrograman untuk belajar algoritma dan pemrograman untuk pemula terutama di bidang akademis. Bahasa ini masuk ke dalam Bahasa tingkat tinggi karena memiliki konsep procedural dan structured programming yang baik. Pascal diambil oleh ahli matematika Pranci yaitu Blaise Pascal yang dikembangkan oleh Niklaus Wirth pada tahun 1970. Kelebihan dari pascal yaitu tipe data yang standar seperti integer, string, Boolean dll memudahkan dalam mendeklarasikan kode. Selain itu, Bahasa ini sederhana dan ekspesif yang strukturnya mendekati Bahasa manusia yaitu Bahasa inggris.

5. JAVA

Yaitu Bahasa pemrograman yang mampu dijalankan di berbagai platform tanpa perlu penyesuaian ulang di paltformnya. Bahasa ini masuk ke dalam Bahasa tingkat tinggi, karena dapat dijalankan dimana saja dan memiliki sintaks yang disusun Java Virtual Machine (JVM) menjadi kode numeric (bytescode). Kelebihan java yaitu memungkinkan programmer dapat menulis kode program secara cepat, Bahasa yang digunakan sederhana, bisa digunakan di sistem operasi manapun dsb.

C. Instalasi Bahasa Pemrograman Pascal

Untuk pemula dalam menggunakan Pascal disarankan menggunakan aplikasi Free Pascal atau Turbo Pascal

  • 1. Untuk menggunakan Free Pascal terlebih dahulu harus mendwonload pada alamat resmi https://freepascal.org ( Pada halaman download ditampilkan dengan memiliki sistem operasi yang akan digunakan, oleh karena itu harus disesuaikan oleh sistem operasi komputer/PC atau laptop kita) . terdapat dua pilihan yaitu :

    • Intel /i36 = windows 32-bit

    AMD64/Intel64/x86_64 = Windows 64-bit

  • 2. Yang kita contohnya menggunkan Intel/i386 = Windows 32-bit. Compiler Free Pascal 32-bit ini berjalan sempurna di sistem 64-bit maupun 32-bit.
  • 3. Pilih alamat mirror untuk memulai proses download.pada halaman sourceForge klik untuk mendownload aplikasi Free Pascal. Halaman link untuk mendownload sebagai berikut:

    • https://sourceforge.net/projects/freepascal/files/latest/download

    Berikutnya, instalasi Free Pascal caranya sama seperti aplikasi pada umnya dengan menambahkan beberapa settingan opsional untuk pembuatan kode program dengan Free Pascal lebih user friendly.

  • 1. Klik dua kali untuk memulai aplikasi proses instalasi
  • 2. Tampilan kedua pada tulisan “Select Destination Location” untuk mengatur lokasi folder di dalam komputer pada instalasi Free Pascal. Di bawah ini secara default akan digunakan berada di C:\FPC\2.6.4. untuk merubah lokasi folder dapat juga C:\FreePascal.
  • 3. Tampilan berikutnya “Select Components” untuk memilih jenis instalasi Free Pascal yaitu Full Instalation, Minimum Instalation atau Custom Instalation. Yang kita contoh yaitu Full Instalation, klik next untuk melanjutkan
  • 4. Tampilan berikutnya “Select Start Menu Folder” untuk mengubah nama shortcut Free Pascal untuk mengakses melalui Start Menu. Klik next saja
  • 5. Dengan memilih pilihan Associate .pas files with Free Pascal IDE, maka ketika kita men-double klik file tersebut dari Windows Explorer, kode program akan langsung ditampilkan di dalam Free Pascal. Anda boleh memilih pilihan ini agar lebih mudah ketika mengedit kode program. Klik Next untuk melanjutkan.
  • 6. Tampilan terakhir akan ditampilkan sebagai berikut, klik finish

D. Instalasi Java Netbeans

Dalam java menggunakan JDK dan JRE. Perbedaannya yaitu JDK (Java Development Kit) adalah software yang digunakan ketika melakukan kompilasi yang berasal kode java menuju bytecode yang mampu dipahami dan mampu dijalankan oleh JRE. JDK memuat program – program yang didalamnya. Sedangkan JRE (Java Runtime Environment) adalah sebuah software yang dipakai dalam sebuah aplikasi yang digunakan java.

Untuk menginstall netbeans kita contohnya langsung satu paket instalasi.

  • 1. Download Java SE dan NetBeans Cobunder (JDK 8u141 and NB 8.2). harus diketahui sistem operasi yang ada di dalam komputer/PC kita apabila 32 bit maka yang diinstall adalah JDK Windows x86, sedangkan apabila 64 bit maka yang diinstall adalah JDK windows x64
  • 2. Selanjutnya, klik dua kali maka tampilan akan seperti di bawah. Tunggu hingga selesai
  • 3. Pilih I accept the term the licence agreement dan setelah itu klik next
  • 4. Pilih tempat JDK dan NetBeans. Klik next
  • 5. Apabila menginginkan update centanglah check for updates , kemudian memulai menginstall klik install
  • 6. Kemudian klik finish untuk menampilkan NetBeans
  • 7. Hingga aplikasi Java seperti di bawah ini
