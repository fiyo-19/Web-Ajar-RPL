A. Lingkungan Kerja Free Pascal

Setelah mendownload dan menginstall Free Pascal, tampilan awal Free Pascal akan seperti gambar di bawah ini .

Free Pascal memiliki beberapa menu utama dan fungsi masing â€“ masing sebagai berikut.

1. File

  • New, untuk membuat program baru
  • New from template, memanggil template contoh yang disediakan oleh Free Pascal

B. Struktur Pemrograman Pascal

Struktur pemrograman pascal dibagi menjadi tiga sebagai berikut.

1. Judul Program

Berisi penanda yang megidentifikasikan sebagai nama program menjelaskan tentang fungsi dan proses yang dikerjakan oleh program.

Format umum : Program Nama_program;

Contoh : Program Konversi_suhu;

2. Deklarasi

Bagian ini dibagi dari beberapa identifier, yaitu diantaranya:

  • Variabel

    • Bagian ini berupa nilai yang bisa berubah sepanjang alur program. Contohnya membuat variabel jari-jari yang diatur bertipe data integer. Nilai jari-jari langsung diberi nilai 8.

  • Fungsi

    • Sebuah fungsi disebut sebagai subprogram, karena dapat memecah alur program yang Panjang menjadi fungsi-fungsi yang saling terpisah. Contohnya function luas_segitiga yang digunakan sebagai menghitung luas segitinga

  • Prosedur

    • Prosedur sebagai tidak mengembalikan nilai. Prosedur sendiri yaitu sesuatu hal unik dalam pascal. Contohnya dengan procedure kel_segitiga dengan menampilkan hasilnya dengan writeln.

  • Main program

    • Sebuah tempat dimana kode program utama ditulis, maka dilakukan pemanggilan fungsi atau prosedur serta pembuatan proses alur kerja program. Main program diawali dengan keyword begin dan diakhiri dengan end. Biasanya program dengan perintah clrscr yang kepanjangannya clear screen untuk menghapus output dari kode program sebelumnya.

3. Badan Program

Bagian ini semua kode akan ditulis diperiksa dan dikerjakan prosesnya oleh compiler dan ditampilkan output pada piranti keluaran. Bagian ini biasanya hanya terdiri dari sebuah pernyataan saja dengan statement part yaitu bagian terakhir dari suatu program. Diawali dengan kata cadangan reserved word yaitu begin dan diakhiri dengan kata cadangan end.

C. Kaidah Penulisan Program Pascal

Untuk penulisan program pada pascal memiliki kaidah sebagai berikut.

  • 1. Di tiap akhir penulisan baris kode program pascal selalu diakhiri tanda titik koma (; )
  • 2. Untuk penulisan komentar pada baris kode program sebagai penjelasan tentang kode program diikuti tanda kurung kurawal (isi komentar) atau dengan tanda {*isi komentar*}
  • 3. Penulisan baris kode program dapat dituliskan dalam satu baris dengan catatan setiap akhir baris dipisahkan dengan tanda titik koma ( ;). Namun, untuk keteraturan dan kerapian dalam pengkodingan sebaiknya setiap baris selalu dipisahkan dalam baris baru untuk mengantisipasi perbaikan kesalahan debug.
  • 4. Bersifat non case sensitive atau tidak membedakan huruf besar dan huruf kecil dalam hal penulisan perintah
  • 5. Reserve word, Daftar kata (word) yang tidak boleh digunakan sebagai identifier dalam pemrograman misalnya digunakan sebagai konstanta, variabel, dan tipe data bentukan. Reserve word sendiri yaitu istilah yang sudah dideskripsikan dalam compiler sebagai sebuah sintaks khusus dalam pemrograman. Berikut contoh reserved word digunakan dalam Pascal

D. Unit Standar Pascal

  • 1. Crt

    • Digunakan sebagai manajemen input output pada layer monitor teks seperti penjedelaan posisi teks, warna teks, posisi kursor, dsb.

  • 2. DOS

    • Digunakan sebagai fasilitas pengaksesan terhadap prosedur dan fungsi standar dalam DOS Prompt seperti DetTime, SetTime, DiskSize, DiskFree dll.

  • 3. System

    • Unit ini tidak perlu dideklarasikan pada program menggunakan perintah user system karena unit ini merupakan unit default pada pascal yang secara otomatis akan mengenali setiap perintah.

  • 4. Graph

    • Unit yang menyediakan fungsi-fungsi perintah pemrograman proses grafis pada sistem operasi..

E. Struktur Pemrograman Java

Bahasa pemrograma java berbeda dengan pascal yang cenderung ber-platform windows, java mampu beekrja pada multiplatform sistem operasi baik itu linux maupun windows. Salah satu jenis pemrograman tingkat tinggi yang bersifat open source buatan Sun Microsystem yang dilanjutkan oleh Oracle, java memiliki keunggulan yaitu mampu mengeksekusi kode biner hasil kompilasi file program dengan JRE (Java Runtime Environment).

Pada awalnya java seperti dengan pascal adalah tipe Bahasa pemrograman yang menggunakan perintah berbasis teks dengan ekstensi java. File sumber kode program java dapat diedit melalui beberapa aplikasi seperti NetBeans, Notepad, Wordpad, Eclips, Android Studio, dsb.

Pemrograman java merupakan Bahasa yang mendukung pemrograman berorientasi objek secara penuh. Objek dalam pemrograman java dapat dikenali dalam kehidupan sehari â€“ hari. Seperti mobil, motor, dan peswat. Analogi objek dalam kehidupan sehari-hari dapat diterapkan dalam OOP dalam java. Katakteristik dan keadaan objek program dinyatakan dalam fields sedangkan perilaku atau pekerjaan yang dilakukan dinyatakan dalam bentuk methods.

Untuk menulis kode program java harus memperhatikan beberapa hal diantaranya:

  • 1. Case sensitive, di setiap baris kode program dalam java sangat memerhatikan huruf besar dan kecil seperti halnya variabel Tester berbeda dengan tester
  • 2. Dalam menulis nama class program java harus diawali dengan huruf kapital
  • 3. Pemberian nama method dalam class program java diawali dengan huruf kecil
  • 4. Baris komentar tunggal diawali dengan //
  • 5. Baris komentar yang lebih dari satu diawali dengan tanda /* dan diakhiri dengan tanda */
  • 6. Penulisan dan penyimpanan nama file java harus sama dengan nama class
  • 7. Eksekusi program java dimulai dari main method yang diawali dengan baris kode public static void mainstring args[]}
